Usage: woaru docu pro [options]

Generate technical TSDoc/JSDoc documentation for developers

Options:
  -p, --path <path>                Project path (default:
                                   "/Users/halteverbotsocialmacpro/Desktop/arsvivai/WOARU(WorkaroundUltra)")

  --git [branch]                   Document only changes compared to branch
                                   (default: "main")

  --force                          Skip interactive confirmation
  --preview                        Preview changes without writing to files
  -h, --help                       display help for command
